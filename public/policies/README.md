# Church Policies PDF Files

This folder contains the official church policy documents in PDF format.

## How to Add PDF Files:

1. **Place your PDF files in this folder** with these exact names:
   - `membership-guidelines.pdf`
   - `conduct-behavior.pdf`
   - `children-youth-ministry.pdf`
   - `financial-policies.pdf`
   - `privacy-confidentiality.pdf`

2. **File Requirements:**
   - Use PDF format only
   - Keep file sizes reasonable (under 10MB each)
   - Use clear, readable fonts
   - Ensure proper formatting

3. **After adding files:**
   - The buttons on the policies page will automatically become active
   - Users can click "View PDF Document" to open each policy
   - PDFs will open in a new tab/window

## Security Features:

- PDFs inherit the same protection as the policies page
- Right-click is disabled on the policies page
- Users cannot easily download or save the PDFs
- Content is protected from screenshots and copying
